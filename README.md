# random-table-generator

## これはなんですか

Wikipedia から記事タイトルをいくつかガチャって並べるやつです。

https://co3k.github.io/random-table-generator/index.html

## 動作環境

- Node.js v10+
- Yarn

## うごかす

```
$ make server
```

## テスト

```
$ make test
```

## デプロイ (ローカルからやる場合)

※fork した場合は package.json の homepage をうまく書き換える必要がたぶんある

```
$ make deploy
```
